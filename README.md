# Medical Triage Bot

A Python-based terminal application that triages patient symptoms and assigns emergency priority levels, bridging computer science logic with pre-hospital EMS protocols!

## Features
- **Object-Oriented Design:** Uses classes for clean data mapping.
- **Dynamic Vitals Overrides:** Automatically shifts a patient's emergency tier based on clinical metrics like Heart Rate and Blood Pressure.
- **Input Validation:** Prevents software crashes from typo errors.
